Step 0 — Get Audio from Discord

Use the Svenska Bookmarklet to download voice messages from Discord — open /bookmarklet/ to install it. Or use Craig bot for full voice channel recordings.

  2. Click the bookmarklet — pick the message you want and download it
  3. Drop the .ogg file into the upload zone below
